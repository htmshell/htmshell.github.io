HTMShell5

A vocabulary for HTMShell

Recommendation 29 September 2015

Proposal

We propose the 5th major revision of the core language of Super Mario Maker: the Hinttext Mario Shell (HTMShell). In this version, new features are introduced to help level authors, new elements are introduced based on research into prevailing authoring practices, and special attention has been given to defining clear conformance criteria for designers in an effort to improve interoperability.

Core features

  • Generator: Warp zone pipe with access blocked
  • Duration of 2 seconds
  • Static enemy entities

Best Practices

  • No international localization, use universal symbols
  • Rapid comprehension
  • Observe block limit
